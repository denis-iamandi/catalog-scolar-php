# Catalog Școlar PHP — Teza de Licență Aplicație web pentru instituțiile preuniversitare din Republica Moldova. Această aplicație permite gestionarea elevilor, profesorilor, orarului, absențelor și notelor. ## Funcționalități - Formular elev - Orar și profesori - Export Excel pentru orar și absențe - Evidență note - Rapoarte și prezentare aplicație ## Tehnologii utilizate - PHP - MySQL - HTML, CSS, JavaScript - Apache Server ## Structura proiectului - `public_html/` — fișierele vizibile pe server - `config/` — conexiune la baza de date - `sql/` — exportul bazei de date - `docs/` — documentație academică (ERD, UML, prezentare) ## Autor Denis Iamandi — Teza de licență, 2026
